# Quotes Web Crawler

Este README servirá como guia para a execução desta aplicação passo a passo, 
desde o setup inicial ao uso da mesma.

## Ambiente

Para a configuração do ambiente de execução serão necessarias as seguintes dependencias: 

* Ruby 3.0.3

* MongoDB
